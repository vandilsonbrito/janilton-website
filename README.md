# Prof. Janilton Almeida Website

O Professor Janilton é doutor em Ciências e professor de Geografia. O website é um espaço para o professor divulgar seus artigos e vender planos de aula em formato de e-book, além de divulgar o seu trabalho e canal do YouTube.

<h2>Technologies</h2>

- React
- React-Router-Dom
- Tailwind CSS
- Prismic CMS
- YouTube API
- Static Forms
- Swiper JS
- React-Scroll

<h2>Acquired Experience</h2>

Pude praticar roteamento de páginas com a biblioteca react-router-dom e scroll da página com a biblioteca react-scroll. Aprendi a usar o Prismic, um Content Management System, no qual o professor poderá escrever seus artigos através da interface do site do Prismic e os artigos serão publicados automaticamente no site devido a integração da API do Prismic com o seu site. 

Além disso, o site conta com a API do Youtube integrando o canal do Professor Janilton do YouTube, no qual é visto um vídeo introdutório do canal e um botão para direcionar o usuário para o canal do YouTube. 

Na seção de contato é possível o usuário mandar um e-mail diretamente para o professor facilitado pelo Static Forms, o professor receberá a mensagem e ao clicar em 'responder' no seu provedor de e-mail será direcionado para o envio da resposta com o endereço de e-mail do usuário já preenchido otimizando o processo de resposta.
